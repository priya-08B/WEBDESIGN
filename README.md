# WEBDESIGN

Responsive Website using HTML & CSS

This project is a simple responsive website built using HTML5 and CSS3. It demonstrates layout design, styling, and basic webpage structure.


Features

Responsive webpage layout

Navigation bar

Hero / Header section

Content sections

Social media icons

Footer

Styled using external CSS


Technologies Used

HTML5 – Structure of the website

CSS3 – Styling and layout

Font Awesome – Icons
